# Card Game App

Full-stack multiplayer card game server with real-time gameplay, AI bots, and advanced analytics.

A learning project that brings the classic Five Hundred card game to the modern web, featuring event-sourced architecture for perfect game replays and AI opponents.

## Features (Planned)

- **Real-time multiplayer** via WebSockets
- **AI-powered bots** with machine learning capabilities
- **Event-sourced architecture** enabling perfect game replays
- **Detailed statistics** and game analysis tools
- **Multiple card games** (starting with Five Hundred)

## Tech Stack

- **Backend**: Python with event sourcing
- **Frontend**: React
- **Real-time**: WebSocket connections

## Project Status

🚧 **Work in Progress** - Currently implementing the core game engine and CLI interface.

## About Five Hundred

Five Hundred is a trick-taking card game popular in some regions but rarely available online. This project aims to make it accessible to players worldwide while showcasing modern software architecture patterns.
